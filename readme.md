# Readme

- How to start servers

  ```
  chmod -R 755 ./docker (if can't run)
  $ ./docker/start
  $ ./docker/restart
  $ ./docker/stop
  ```

- User Account Mysql

  ```
    Host: mysql
    User: root
    Pass: root
  ```

- Web Server -> http://localhost:8021
- phpMyAdmin -> http://localhost:8025
- Adminer -> http://localhost:8026
